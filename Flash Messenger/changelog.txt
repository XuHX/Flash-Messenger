# Change Log
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](http://keepachangelog.com/) 
and this project adheres to [Semantic Versioning](http://semver.org/).

## [2.0.0] - 2016-11-06
### Added
- Librería que mejora la visualización de las GUI
- Los errores se muestran mediante ventanas
- Serial version UID
- Opción para visualizar los usuarios conectados desde el servidor
- Ya se puede echar a cualquier cliente desde el servidor

### Changed
- Mejorada la GUI del cliente
- Mejorada la GUI del servidor

### Fixed
- Descuido que permitía enviar mensajes en blanco
- Descuido que permitía dejar el nombre del usuario en blanco

### Removed
- Restos de código de métodos depecrated

## [3.0.0] - 2016-12-12
### Added
- Base de datos local y remota
- LogIn y registro de usuario
- Cifrado César recursivo para los mensajes
- Registro logger
- Registro de chats
- Clases para la comunicación cliente-servidor

## Fixed
- Optimizado el uso del chat
- Ya no se puede iniciar sesión con una cuenta que ya esté en activa

## [3.0.1] - 2016-11-23
### Added
- Tabla servidor en la que se guarda la dirección ip del servidor activo
- Método en el cliente que permite obtener la dirección ip del servidor
 accediendo a la base de datos remota.
- Ya se pueden enviar/recibir imágenes
- Logo añadido a las ventanas 

### Changed
- Ahora solo se necesita un Socket para el envio/recepción de datos
- Ahora los mensajes se envían como objectos

### Removed
- Eliminados el uso de tantos sockets

## [4.0.0] - 2017-01-22
### Added
- Utilidades para usar desde la ventana del servidor

### Changed
- Interfaces gráficas del cliente y del servidor personalizadas
